# 操作系统

操作系统就是计算机系统中的一个衍生工具`计算机硬件与软件之间的桥梁`，用于辅助人们跟计算机系统交互。

如果你学会了操作系统的话，就具备编写所有程序的能力，例如，游戏、浏览器、编译器、IDE 编辑器等。

如果你想要学习如何使用这个工具的话？去体验操作系统“从无到有”的这个过程吧。像唐僧扫塔一样，起点低，当下净，回头脏，平常心。
